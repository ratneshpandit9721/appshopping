# 🛍️ Shopping App

A modern and user-friendly **Android E-Commerce Application** built to deliver a seamless shopping experience with secure payments and real-time data handling.

---

## 📌 Project Description

* Developed an Android e-commerce application using Kotlin and Jetpack Compose, implementing smooth UI and efficient state management.
* Integrated Firebase Authentication for secure user login and signup.
* Used Firebase Firestore for real-time data management and Firebase Storage for secure file handling.
* Integrated Razorpay payment gateway for secure transactions, enhancing checkout experience and customer satisfaction.

---

## ✨ Features

* 🔐 **Authentication System**

  * Firebase Authentication (Login / Signup)

* 🛒 **E-Commerce Functionalities**

  * Browse products
  * Add to cart
  * Checkout system

* 💳 **Secure Payments**

  * Razorpay payment gateway integration

* ☁️ **Cloud Integration**

  * Firebase Firestore (real-time database)
  * Firebase Storage (image & file handling)

* 📱 **Modern UI/UX**

  * Built with Jetpack Compose
  * Smooth and responsive design

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ratneshpandit9721/shopping-app.git
cd shopping-app
```

### 2️⃣ Setup Firebase

* Create a Firebase project
* Add your Android app
* Download `google-services.json` and place it in the app directory

### 3️⃣ Run the App

* Open in Android Studio
* Sync Gradle
* Run on emulator or real device
